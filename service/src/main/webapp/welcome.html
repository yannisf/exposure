Welcome to Perfect exposure!

Perfect exposure is a web application that provides tools to help you achieve the elusive perfect exposure. Currently, it consists of three modules.

Equivalent

The user selects an initial exposure, typically one the camera light meter suggests. Then alters one of the exposure parameters and the application compensates to achieve an equivalent exposure.

Filter

The user selects an initial shutter speed, typically suggested by the camera. Then selects and stacks filters. The application provides the facility to compensate the filtering by adjusting the shutter speed.

DOF

The user provides initial exposure parameters, such as sensor type, focal length and subject distance. The application calculates important focus parameters such as hyperfocal distance, near and far focus points.
